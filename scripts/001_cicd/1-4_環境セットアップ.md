# セクション1 レクチャー4: 開発環境のセットアップ

## 動画情報
- **時間**: 約5分
- **形式**: 画面共有（ハンズオン）

---

## 台本

### オープニング（0:00-0:20）

このレクチャーでは、
コースを進めるために必要な環境をセットアップします。

すでに準備ができている方は、
次のセクションに進んでいただいて大丈夫です。

---

### 必要なもの一覧（0:20-1:00）

このコースで必要なものは以下の通りです。

**必須:**
1. AWSアカウント
2. GitHubアカウント
3. Git（ローカル環境）
4. テキストエディタ（VS Code推奨）

**あると便利:**
5. AWS CLI
6. SAM CLI（セクション7で使用）
7. Node.js（テスト用）

---

### AWSアカウント（1:00-2:00）

AWSアカウントをお持ちでない方は、
aws.amazon.com から作成してください。

**注意点:**
- クレジットカードの登録が必要です
- 無料利用枠を活用すれば、このコースの内容は
  月額数ドル程度で実践できます
- 使い終わったリソースは削除することを忘れずに

アカウント作成の詳細は、
AWSの公式ドキュメントを参照してください。

---

### GitHubアカウント（2:00-2:30）

GitHubアカウントをお持ちでない方は、
github.com から作成してください。

無料アカウントで十分です。

このコースでは、GitHubをソースコードの
管理場所として使用します。

---

### Gitのインストール（2:30-3:30）

ローカル環境にGitをインストールします。

**Windows:**
Git for Windowsをダウンロードしてインストール
https://gitforwindows.org/

**Mac:**
```bash
# Homebrewを使う場合
brew install git

# または Xcode Command Line Tools
xcode-select --install
```

**インストール確認:**
```bash
git --version
# git version 2.x.x と表示されればOK
```

---

### テキストエディタ（3:30-4:00）

テキストエディタは何でも構いませんが、
VS Codeをおすすめします。

**VS Codeのメリット:**
- 無料
- Git統合
- AWS関連の拡張機能が豊富
- ターミナル内蔵

https://code.visualstudio.com/

---

### AWS CLI（任意）（4:00-4:30）

AWS CLIは必須ではありませんが、
あると便利です。

**インストール:**
https://aws.amazon.com/cli/

**設定:**
```bash
aws configure
# Access Key ID: [IAMで作成したキー]
# Secret Access Key: [IAMで作成したシークレット]
# Default region: ap-northeast-1
# Default output format: json
```

セクション7のSAMで使用します。

---

### クロージング（4:30-5:00）

環境のセットアップは以上です。

準備ができたら、次のセクションで
S3静的サイトホスティングを始めましょう！
