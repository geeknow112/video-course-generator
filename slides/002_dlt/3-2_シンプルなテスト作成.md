# シンプルなテストの作成と実行

## ハンズオン

---

# テスト作成画面

左メニュー → 「Create Test」

---

# 基本情報

| 項目 | 設定値 |
|------|--------|
| Name | my-first-load-test |
| Description | 初めての負荷テスト |

---

# テスト設定

| 項目 | 設定値 | 説明 |
|------|--------|------|
| Task Count | 1 | Fargateタスク数 |
| Concurrency | 10 | 1タスクあたり同時接続数 |
| Ramp Up | 30秒 | 負荷を上げる時間 |
| Hold For | 60秒 | 最大負荷維持時間 |

**→ 1タスク × 10同時接続 = 10同時ユーザー**

---

# ターゲット設定

| 項目 | 設定値 |
|------|--------|
| HTTP Method | GET |
| Target URL | https://httpbin.org/get |

※httpbin.org = テスト用の公開API

---

# テスト実行

1. 「Run Test」クリック
2. 確認ダイアログで「Confirm」
3. ステータス: `Running`

⏱️ Fargateタスク起動まで少し待機

---

# 結果確認

| メトリクス | 意味 |
|------------|------|
| Avg Response Time | 平均レスポンスタイム |
| Requests per Second | 秒間リクエスト数 |
| Success Rate | 成功率 (100% = 全成功) |
