# ログイン手順

1. CloudFormation出力のConsoleUrlにアクセス
2. メールアドレス入力（AdminEmail）
3. 一時パスワード入力（メールで届いたもの）
4. 「サインイン」クリック

---

# パスワード変更

**要件:**
- 8文字以上
- 大文字・小文字・数字・記号を含む

例: `MyTest@2024!`
