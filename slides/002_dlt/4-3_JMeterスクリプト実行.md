# JMeterスクリプトの実行

## ハンズオン

---

# スクリプトアップロード

1. DLT Webコンソール → Create Test
2. Test Type: **JMeter** を選択
3. Upload Script → `api-load-test.jmx` を選択

---

# テスト設定

| 項目 | 設定値 | 結果 |
|------|--------|------|
| Name | jmeter-api-test | - |
| Task Count | 5 | 5タスク |
| Concurrency | 20 | ×20同時接続 |
| Ramp Up | 60秒 | - |
| Hold For | 300秒 (5分) | - |

**→ 5 × 20 = 100同時ユーザー**

---

# 負荷の比較

| テスト | 同時ユーザー |
|--------|--------------|
| 前回（シンプル） | 10 |
| 今回（JMeter） | **100** |

10倍の負荷をかけてシステムの挙動を確認

---

# テスト実行

1. 「Run Test」クリック
2. 「Confirm」で確定
3. 5つのFargateタスクが起動
4. リアルタイムでメトリクス表示

---

# 結果確認ポイント

✅ 100ユーザーでのレスポンスタイム推移
✅ p99の値は許容範囲内か
✅ エラーは発生していないか
✅ 10ユーザーとの比較
