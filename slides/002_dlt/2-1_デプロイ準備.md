# デプロイ準備

## CloudFormationでDLTをデプロイ

約15分で環境構築完了

---

# 事前に必要なもの

**1. AWSアカウント**
- 管理者権限が必要

**2. VPCとサブネット**
- インターネットアクセス可能なサブネット
- パブリックサブネット or NAT Gateway付きプライベート

**3. メールアドレス**
- ログイン用（一時パスワードが届く）

---

# VPC/サブネット要件

```
インターネット
     ↓
┌────────────────────────────────────┐
│ VPC                                 │
│  ┌─────────────────────────────┐   │
│  │ パブリックサブネット         │   │
│  │   → そのまま使える ✅       │   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │ プライベートサブネット       │   │
│  │   → NAT Gateway必須 ⚠️      │   │
│  └─────────────────────────────┘   │
└────────────────────────────────────┘
```

# リージョンの選択

**推奨: ターゲットシステムに近いリージョン**

| リージョン | 用途 |
|------------|------|
| ap-northeast-1 (東京) | 日本向けシステム |
| us-east-1 (バージニア) | 北米向けシステム |
| eu-west-1 (アイルランド) | 欧州向けシステム |

※マルチリージョンテストも可能
