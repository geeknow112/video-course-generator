# S3とは

**Simple Storage Service** の略

- AWSのオブジェクトストレージサービス
- ファイルを保存・配信できる
- 高い耐久性（99.999999999%）
- 低コスト

---

# S3の用途

- **静的ウェブサイトのホスティング** ← 今回はこれ
- バックアップ・アーカイブ
- データレイク
- アプリケーションのアセット配信
- ログの保存

---

# S3の基本用語

| 用語 | 説明 |
|-----|------|
| **バケット** | ファイルを入れる箱（フォルダのようなもの） |
| **オブジェクト** | バケット内のファイル |
| **キー** | オブジェクトの名前（パス） |
| **リージョン** | バケットを作成する場所 |

---

# バケットの特徴

- バケット名は **世界で一意** である必要がある
- 一度作成したら名前は変更できない
- リージョンを指定して作成
- バケット内にフォルダ構造を作れる

---

# 静的ウェブサイトホスティング

S3バケットをウェブサーバーとして使う機能

## できること
- HTML/CSS/JavaScriptの配信
- 画像・動画の配信
- シングルページアプリケーション（SPA）

## できないこと
- サーバーサイド処理（PHP, Python等）
- データベース接続
