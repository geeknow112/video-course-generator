# 収録手順

1. Google Slidesで上記スライドを作成
2. スライドショー開始（F5 または Ctrl+F5）
3. `Win + Alt + R` で録画開始
4. 台本（1-1_コース紹介.md）を見ながら喋る
5. スライドは矢印キーで送る
6. 終わったら `Win + Alt + R` で停止
7. `ビデオ` フォルダにMP4が保存される

目安時間: 5分
