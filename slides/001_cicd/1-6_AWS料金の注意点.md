# AWS無料利用枠

## 12ヶ月無料枠（アカウント作成から1年間）
- EC2: 月750時間（t2.micro）
- S3: 5GB
- Lambda: 月100万リクエスト

## 常時無料枠
- Lambda: 月100万リクエスト
- DynamoDB: 25GB

# このコースでかかる費用

| サービス | 費用目安 |
|---------|---------|
| CodePipeline | $1/月（1つ目は無料） |
| CodeBuild | 月100分まで無料 |
| S3 | 〜$0.50/月 |
| Lambda | ほぼ無料 |

**合計: 月額 $2〜3（300〜450円）程度**

---

# 課金が発生するケース

## 注意が必要なパターン

1. **リソースの削除忘れ** - 一番多い原因
2. **無料枠の超過** - 使いすぎ
3. **リージョン間違い** - 東京以外で作成
4. **本番用の設定** - 高スペックを選択

---

# 課金を防ぐポイント

## 1. 予算アラートを設定
- AWS Budgetsで月額上限を設定
- 超えそうになったらメール通知

## 2. ハンズオン後は必ず削除
- 使い終わったらすぐ削除
- 「後で」は忘れる原因

## 3. リージョンを確認
- 東京リージョン（ap-northeast-1）を使用

---

# 予算アラートの設定方法

1. AWSコンソール → 「Billing」
2. 「Budgets」→「予算を作成」
3. 月額予算を設定（例: $10）
4. 80%到達でメール通知

**最初に設定しておくと安心です**

---

# リソース削除のチェックリスト

ハンズオン後に確認：

- [ ] S3バケット（中身も削除）
- [ ] CodePipelineパイプライン
- [ ] CodeBuildプロジェクト
- [ ] Lambda関数
- [ ] IAMロール（不要なもの）
- [ ] CloudWatchロググループ

---

# まとめ

- 無料利用枠を活用すれば **月数百円** で学習可能
- **予算アラート** を最初に設定
- ハンズオン後は **必ずリソース削除**
- 不安なときは **コスト管理画面** を確認

**安心して学習を進めましょう！**
