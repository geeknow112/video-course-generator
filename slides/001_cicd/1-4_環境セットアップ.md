# 必要なもの一覧

## 必須
- ✅ AWSアカウント
- ✅ GitHubアカウント
- ✅ Git（ローカル環境）
- ✅ テキストエディタ（VS Code推奨）

## あると便利
- ⬜ AWS CLI
- ⬜ SAM CLI（セクション7で使用）

---

# AWSアカウント

**作成方法**: https://aws.amazon.com/

## 注意点
- クレジットカードの登録が必要
- 無料利用枠で月額 **数ドル程度**
- 使い終わったリソースは **必ず削除**

詳細はAWS公式ドキュメントを参照

---

# GitHubアカウント

**作成方法**: https://github.com/

- 無料アカウントでOK
- ソースコードの管理場所として使用
- CodePipelineと連携

---

# Gitのインストール

## Windows
Git for Windows をダウンロード
https://gitforwindows.org/

## Mac
```bash
brew install git
# または
xcode-select --install
```

## 確認
```bash
git --version
```

---

# テキストエディタ

**VS Code** をおすすめします

https://code.visualstudio.com/

## メリット
- 無料
- Git統合
- AWS関連の拡張機能が豊富
- ターミナル内蔵

# AWS CLI（任意）

セクション7のSAMで使用します

**インストール**: https://aws.amazon.com/cli/

## 設定
```bash
aws configure
# Access Key ID: [IAMで作成]
# Secret Access Key: [IAMで作成]
# Default region: ap-northeast-1
# Default output format: json
```
